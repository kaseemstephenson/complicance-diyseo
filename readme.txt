== Important Notice: External Services ==

DIYSEO utilizes external services to provide its functionality:

1. License Validation and Content Generation:
   - Our plugin communicates with our servers (https://diyseoapi-69f00dabdd4f.herokuapp.com) for license validation and content generation.
   - Your license key and content prompts are sent to these servers for processing.
   - No user data is stored on our servers beyond the processing of individual requests.

2. OpenAI API:
   - DIYSEO uses OpenAI's API (https://api.openai.com) for AI-powered content generation and image creation.
   - Your content prompts are sent to OpenAI's servers for processing.
   - Please review OpenAI's terms of service (https://openai.com/policies/terms-of-use) and privacy policy (https://openai.com/policies/privacy-policy) for more information.

A DIYSEO account and valid license key are required to use these features. By using DIYSEO, you agree to the transmission of necessary data to these external services for the plugin to function.

== Installation ==

1. Upload the `diyseo` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Obtain a license key from diyseo.ai
4. Enter your license key in the DIYSEO settings page
5. Start using DIYSEO features in your post editor!


== Frequently Asked Questions ==

= Do I need a license key to use DIYSEO? =

Yes, a valid license key is required to use DIYSEO. Without a license key, the plugin features will not be available.

= How do I get a license key? =

You can obtain a license key by visiting diyseo.ai and obtaining a DIYSEO subscription(free tier available).

= Can I use DIYSEO with any WordPress theme? =

Yes, DIYSEO is designed to work with any WordPress theme. However, if you encounter any compatibility issues, please contact our support team.

= Is the generated content unique? =

Yes, the content generated by DIYSEO is unique for each request. However, as with any AI-generated content, we recommend reviewing and editing the output to ensure it meets your specific needs and style.

= How do I generate content using DIYSEO? =

After activating the plugin and entering your license key, you'll see new meta boxes in your post editor. Simply fill in the required fields (like post title and desired word count) and click the "Generate" button to create your content.
